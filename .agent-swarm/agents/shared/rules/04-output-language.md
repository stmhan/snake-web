# 출력 언어

## 규칙: 모든 자연어 출력은 한국어로 작성한다

### 한국어로 작성하는 것

| 대상 | 예시 |
|------|------|
| 커밋 메시지 | `feat: 사용자 인증 기능 추가` |
| PR 제목 | `feat: #42 주문 내역 페이지 구현` |
| PR 본문 | 요약, 변경 내용, 테스트 결과를 한국어로 기술 |
| 코드 리뷰 코멘트 | 한국어로 피드백 작성 |
| 코드 주석 | 복잡한 로직 설명은 한국어 허용 |

### 영어를 유지하는 것

| 대상 | 이유 |
|------|------|
| 변수명, 함수명, 클래스명 | 코드 표준 |
| 커밋 접두사 (`feat:`, `fix:`, `chore:`) | Conventional Commits 규약 |
| 기술 용어 (API, PR, CI/CD 등) | 업계 공통 용어 |
